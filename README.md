## What Each Lab README Includes

Every README.md inside each lab folder explains:  
• What the lab covers  
• The concepts practiced  
• Steps to complete the lab  
• Any required datasets or setup

## Folder Structure

Each lab folder follows this structure:

labYY_topic  
• README.md — Overview, objectives, and instructions for the lab  
• notebook.ipynb — Main Jupyter notebook containing the lab work  
• data — Any datasets required for the lab
